# Json File Editor
Программа предназначена для тестирования ссылок прописанных в файле sitemap. 

Программа циклично проходит по всем ссылкам и получает статус страницы. 
Если статус не равен 200 (успешно) данная ссылка попадет в список ошибок.

Системные требования: 
- Операционная система Windows 7, 8, 10
- Microsoft .NET Framework 4.8
	cкачать .NET Framework Runtime можно по ссылке: https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48

Ссылка на приложение: https://github.com/SomovStudio/TestSitemap
